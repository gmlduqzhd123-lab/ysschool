# YSSCHOOL Learning Games 100 - V5 Mainboard Return Fix

## 수정 내용

- 게임 실행 후 100개 게임 통합 메인보드로 돌아가는 버튼이 사라지거나 작동하지 않는 문제를 수정했습니다.
- 모든 게임에 `🏠 100게임 메인보드` 고정 버튼을 화면 좌측 상단에 표시하도록 했습니다.
- 게임 화면의 오버레이, 전체화면, 모바일 주소창, 하단 버튼 겹침 상황에서도 복귀 버튼이 사라지지 않도록 `z-index: 2147483647`로 고정했습니다.
- 복귀 경로는 다음 순서로 안전하게 판단합니다.
  1. 대시보드가 게임 실행 시 전달한 `return` 파라미터
  2. 브라우저 저장소에 저장된 마지막 대시보드 주소
  3. `../learning-games-all/index.html` 상대경로
- 로컬 `file://` 실행과 Vercel 배포 경로를 모두 고려했습니다.
- 대시보드에서 게임 실행 시 `return` 파라미터를 자동으로 붙이도록 수정했습니다.
- 대시보드 복귀 시 필터와 화면이 사라진 것처럼 보이지 않도록 `pageshow`, `visibilitychange`에서 다시 렌더링하도록 보강했습니다.

## 적용 방법

압축 파일 안의 `public` 폴더 전체를 프로젝트의 `public` 폴더에 병합하세요.

실행 경로:

```text
/learning-games-all/index.html
```
