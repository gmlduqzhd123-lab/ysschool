100개 학습 게임 V10 다이나믹 밸런스판을 기반으로 V11 글꼴 적용판을 반영해줘.

적용 요구사항:
1. 압축 파일 안의 public 폴더 전체를 프로젝트 public 폴더에 병합한다.
2. /learning-games-all/index.html 경로가 유지되어야 한다.
3. public/learning-games-assets/ysschool_cute_fonts.css 파일을 유지한다.
4. public/learning-games-assets/cute_fonts_patch.js 파일을 유지한다.
5. learning-games, learning-games-2부터 learning-games-10까지 모든 게임 HTML과 learning-games-all/index.html에서 위 CSS와 JS가 연결되어 있어야 한다.
6. 기존 게임 기능, 난이도 조정 기능, 메인보드 복귀 기능, 수업 메뉴 기능은 삭제하거나 변경하지 않는다.
7. 배포 후 /learning-games-all/index.html 접속, 게임 3개 이상 실행, 제목/버튼/문제 글꼴 적용 여부를 확인한다.
