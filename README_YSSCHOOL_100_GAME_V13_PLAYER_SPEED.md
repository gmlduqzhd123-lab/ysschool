# YSSCHOOL Learning Game World V13 - 캐릭터 이동 속도 보강판

V12 글꼴 깨짐 수정판을 기준으로, 게임 캐릭터가 직접 이동해야 하는 게임들의 조작 반응 속도를 높였습니다.

## 주요 수정
- 캐릭터 직접 이동 속도 보정 함수 `YS_playerSpeed`, `YS_playerStep`, `YS_laneSnap` 추가
- 매쓰 서바이버즈 캐릭터 이동 속도 상향
- 도형 오비 미로 캐릭터 이동 속도 상향
- 어휘 러너 차선 전환 반응 속도 상향
- 독해 탐정 런 방향키/터치 이동 거리 상향
- 소수 우주 채굴단 키보드 좌우 이동 거리 상향
- 2탄 슈팅형 게임에서 좌우 터치 버튼을 표시하고, 방향키 좌우 이동이 실제 우주선 이동으로 작동하도록 수정
- 적군/장애물 속도는 그대로 두어 전체 난이도가 갑자기 어려워지지 않도록 조정
