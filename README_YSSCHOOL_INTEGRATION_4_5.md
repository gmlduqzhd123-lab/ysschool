# 초등 학습 HTML 게임 20종 추가판: 4탄·5탄

## 배포 경로
아래 두 폴더를 Vercel/Next.js 프로젝트의 `public` 폴더 아래에 그대로 복사합니다.

- `public/learning-games-4` → `/learning-games-4/index.html`
- `public/learning-games-5` → `/learning-games-5/index.html`

## 포함 기능
- Web Audio 기반 브금과 효과음
- 아이템, 콤보, 레벨업, 최고점 저장
- 배움 기록장, 미션 뽑기, 5분 집중 타이머
- 저사양 모드, 전체화면, 모바일 터치 조작
- 홈페이지 복귀 및 게임 목록 복귀 버튼

## 점검
- 모든 JS 파일 `node --check` 문법 검사 통과
- `index.html` 링크와 실제 파일명 대조
- 파일명은 Vercel 정적 배포에 안전한 영문 소문자·숫자·언더바로 구성
