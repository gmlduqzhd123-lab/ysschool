# 엽쌤스쿨 배움게임월드 100종 V7

## 적용 경로
- `/learning-games-all/index.html`

## V7에서 추가한 기능
1. 오늘의 추천 게임
2. 학년별 3게임 추천 코스
3. 별 스티커북 및 묶음별 진행률
4. 즐겨찾기와 최근 플레이
5. 교사용 수업 모드
6. 전체 무음 수업 모드
7. 게임 안 별 저장 버튼
8. 게임 안 10초 복습 카드
9. 연습모드 실행값 전달
10. 기존 100개 게임 형태 유지

## 적용 방법
압축 파일 안의 `public` 폴더 전체를 Vercel 프로젝트의 `public` 폴더에 병합합니다. 기존 1~10탄 게임 폴더와 `learning-games-all`, `learning-games-assets`가 함께 있어야 합니다.

## 확인 사항
- `/learning-games-all/index.html` 접속
- 오늘 추천 게임 실행
- 100게임 메인보드 복귀 버튼 확인
- 별 저장 후 대시보드 스티커북 반영 확인
- 교사용 무음 모드에서 게임 실행 확인
