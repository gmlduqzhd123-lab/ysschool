# 엽쌤스쿨 배움게임월드 100종 V10 다이나믹 밸런스판

## 적용 경로
`/learning-games-all/index.html`

## V10 변경 사항
- V9의 게임 형태와 대시보드 구조는 유지했습니다.
- V9에서 초반 1라운드 적군·장애물 속도가 지나치게 느리게 느껴지던 문제를 조정했습니다.
- 난이도 이름을 다음처럼 정리했습니다.
  - 다이나믹 쉬움: 쉬운 선택지와 넉넉한 생명은 유지하되 속도는 답답하지 않게 조정
  - 균형 보통: 고학년도 재미있게 느낄 정도로 등장 간격과 속도 강화
  - 신나는 도전: 빠르고 역동적인 도전용 난이도
- 공통 엔진 게임의 적군 이동 속도, 떨어지는 카드 속도, 리듬 노트 속도, 정렬 카드 속도, 등장 간격을 보정했습니다.
- 시간이 지날수록 조금씩 빨라지는 완만한 페이스 램프를 추가했습니다.
- 생명, 아이템, 실수 기회는 완전히 줄이지 않아 수업용 안정성은 유지했습니다.

## 적용 방법
압축 파일의 `public` 폴더 전체를 기존 프로젝트의 `public` 폴더에 병합하세요.
기존 파일을 덮어쓰기 하면 됩니다.
