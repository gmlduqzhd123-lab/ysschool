# 엽쌤스쿨 배움게임월드 V9 안정화판

## 적용 내용
- 100개 게임의 기존 게임 형태 유지
- 공통 UI 정리: 게임 안에서는 좌측 상단 `☰ 수업메뉴` 하나로 메인보드/별 저장/복습/다시 시작 관리
- 게임 종료 화면에서 메인보드 복귀 버튼 안정화
- 별 스티커 기준을 단순 접속이 아닌 플레이 시간·정답 시도·점수 기준으로 변경
- 테트리스·머지류 난이도 추가 완화
- 장식용 캐릭터/버디 오버레이 제거
- 브금 중복 레이어 제거, 게임별 사운드 시드만 유지
- 방향키/터치 조작 시 화면 흔들림 방지 범위를 게임 영역 중심으로 정리
- 대시보드 첫 화면 24개 우선 표시 + 더 보기 방식으로 경량화
- 교사 모드에서 무음 수업 모드 권장

## 실행 경로
`/learning-games-all/index.html`

## 적용 방법
압축 파일 안의 `public` 폴더 전체를 프로젝트의 `public` 폴더에 병합하세요. 기존 1~10탄 게임 폴더와 `learning-games-all` 폴더가 모두 함께 있어야 합니다.
