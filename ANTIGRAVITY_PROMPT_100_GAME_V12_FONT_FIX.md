100개 학습 게임의 V11 글꼴 깨짐 문제를 V12 기준으로 수정해줘.

요구사항:
1. public 폴더 전체를 프로젝트 public 폴더에 병합한다.
2. 기존 기능과 게임 파일 구조는 유지한다.
3. public/learning-games-assets/ysschool_cute_fonts.css를 최신 파일로 교체한다.
4. public/learning-games-assets/cute_fonts_patch.js를 최신 파일로 교체한다.
5. Bagel Fat One, Gaegu, Gamja Flower를 기본 글꼴로 사용하지 않는다.
6. 제목은 Jua, 본문/문제는 Noto Sans KR와 Gowun Dodum 중심으로 적용한다.
7. 대시보드와 게임 3개 이상에서 한글 제목, 설명, 문제 글자가 깨지지 않는지 확인한다.
8. 기존 난이도 조정, 사운드, 메인보드 복귀, 수업 메뉴 기능은 건드리지 않는다.
