# YSSCHOOL 초등 학습 HTML 게임 10종 2탄

## 배치 경로
프로젝트 루트 기준으로 아래 폴더를 그대로 복사합니다.

```text
public/learning-games-2/
```

배포 후 접속 주소는 다음과 같습니다.

```text
/learning-games-2/index.html
```

## 포함 게임
1. 맞춤법 카트 레이싱 - 국어 맞춤법과 어휘
2. 분수 피자 타이쿤 - 분수와 동치분수
3. 영어 몬스터 캐처 - 기초 영어 어휘
4. 과학 운석 블래스터 - 과학 개념 분류
5. 역사 타임포털 러너 - 역사 인물과 사건
6. 코딩 로봇 미로 - 순차, 조건, 반복, 디버깅
7. 도형 타워 스택 - 도형 속성
8. 시계 배달 대작전 - 시각과 경과 시간
9. 환경 리듬 히어로 - 환경 실천
10. 좌표 보물지도 - 좌표와 지도 읽기

## 공통 기능
- Web Audio 기반 브금과 효과음
- 아이템: 방패, 자석, 빙결, 더블 점수, 시간 추가, 생명, 폭탄
- 최고점 저장
- 콤보와 레벨업
- 5분 수업 모드
- 저사양 모드
- 배움 기록장
- 미션 뽑기
- 모바일 터치 조작
- 외부 이미지, 음원, CDN 없음

## 주의
브라우저 정책상 사운드는 사용자의 첫 클릭 또는 터치 이후에만 재생됩니다. 오류가 아니라 브라우저 기본 정책입니다.
