# 초등 학습 게임 100종 통합 대시보드 - 원본 게임 형태 유지 난이도 조정판

## 적용 경로

`/learning-games-all/index.html`

## 핵심 수정

- 기존 1~10탄 게임 파일과 장르 구조를 유지했습니다.
- 저학년 학생도 플레이할 수 있도록 속도, 스폰 간격, 체력, 제한 시간, 아이템 확률을 완화했습니다.
- 대시보드에서 학년군과 난이도를 선택하면 각 게임 URL에 `grade`, `difficulty` 값이 전달됩니다.
- 각 게임 화면 하단에 현재 학년·난이도 배지가 표시됩니다.

## 포함 게임 수

총 100개

## Vercel 적용

압축 파일의 `public` 폴더 내용을 프로젝트의 `public` 폴더에 병합하세요.
기존 `/learning-games`, `/learning-games-2` … `/learning-games-10` 경로를 유지합니다.
통합 대시보드는 `/learning-games-all/index.html`입니다.


## V3 수정 사항

- 통합 대시보드에서 게임 실행 시 `ERR_FILE_NOT_FOUND`가 발생하던 문제를 수정했습니다.
- 원인: 대시보드의 게임 링크가 `/learning-games/...` 절대경로로 되어 있어, 로컬 `file://` 환경에서는 컴퓨터 루트 폴더를 찾으려고 했습니다.
- 수정: 대시보드 기준 상대경로 `../learning-games/...` 방식으로 변경했습니다. 이 방식은 로컬 테스트와 Vercel 배포 환경에서 모두 동작합니다.
- 게임 화면 하단의 통합 대시보드 복귀 링크도 `../learning-games-all/index.html`로 수정했습니다.


## V4 추가 보강 내용

- 100개 게임의 기존 형태는 유지한 채, 게임별로 브금과 효과음의 음계·템포·효과음을 다르게 들리도록 보강했습니다.
- 각 게임 화면에 귀여운 버디 캐릭터와 오늘의 버디 배지를 추가했습니다.
- 6~10탄 공통 엔진 게임은 실제 플레이어 아바타도 귀여운 동물 버디로 바뀌도록 사전 패치를 적용했습니다.
- 화살표 키 또는 모바일 방향 버튼 조작 시 브라우저 기본 스크롤이 작동해 화면이 흔들리는 문제를 방지했습니다.
- 대시보드 경로는 그대로 `/learning-games-all/index.html`입니다.
