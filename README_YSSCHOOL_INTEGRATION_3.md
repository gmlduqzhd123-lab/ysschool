# 초등 학습 HTML 게임 10종 3탄

## 배포 경로
`public/learning-games-3/` 폴더를 Vercel/Next.js 프로젝트의 `public` 폴더 안에 그대로 넣으면 됩니다.

접속 주소: `/learning-games-3/index.html`

## 포함 기능
- Web Audio 기반 브금/효과음
- 아이템, 콤보, 레벨, 최고점 저장
- 배움 기록장, 미션 뽑기, 5분 집중 타이머
- 저사양 모드, 전체화면, 모바일 터치 조작
