# 엽쌤스쿨 배움게임월드 100종 V8

## 수정 핵심
- 게임 종료 화면에 `🏠 100게임 메인보드` 복귀 버튼을 추가했습니다.
- 게임 실행 중에도 좌측 상단 고정 버튼으로 메인보드에 돌아갈 수 있습니다.
- `./index.html`로 돌아가서 각 탄 목록으로 빠지는 링크를 통합 메인보드 기준으로 보정했습니다.
- 화면에 떠 있던 불필요한 장식용 버디/캐릭터 오버레이를 제거했습니다.
- 6~10탄 공통 엔진 게임은 게임 목적에 맞는 조작 캐릭터가 움직이도록 캐릭터 표시 로직을 수정했습니다.
- 방향키와 터치 조작 시 화면 흔들림 방지 기능을 유지·보강했습니다.

## 적용 경로
`/learning-games-all/index.html`

프로젝트의 기존 `public` 폴더에 압축 파일 안의 `public` 폴더 전체를 병합하면 됩니다.
